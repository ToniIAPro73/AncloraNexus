Documento de Prueba para Conversión HTML a PDF

Este es un documento de prueba que contiene varios elementos HTML para verificar la conversión a PDF.

Características del documento:

  • Texto con acentos: conversión, información, configuración
  • Caracteres especiales: ¿Cómo está? ¡Excelente!
  • Números y símbolos: 123, $, %, &

Nota importante: Este texto debería aparecer resaltado en el PDF final.
